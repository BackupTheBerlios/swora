Verbindungsaufbau zur Datenbank
Um erfolgreich Verbindung zum SQL-Server herstellen zu können, müssen Sie extakte Angaben über Host, Benutzer, Passwort und den Namen der zu verwendenden Datenbank machen.
Host
Host der Datenbank (meist:LOCALHOST)
UserName
Benutzername für die Datenbank
UserPasswort
BenutzerPasswort für die Datenbank
Datenabank
Name der zu benutzenden Datanbank
